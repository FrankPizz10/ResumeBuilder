# AI Resume Builder

A simple and intuitive AI-powered resume builder that helps users create professional resumes quickly and easily. This project is built with React and TypeScript, utilizing Tailwind CSS for styling and shadcn/ui components.

## Features

- **User-Friendly Interface**: Easy-to-use form for entering personal information, experience, and education.
- **Real-Time Preview**: View a live preview of the resume as you fill out the form.
- **PDF Download**: Future functionality to download the resume as a PDF (coming soon).
- **Responsive Design**: Works seamlessly on both mobile and desktop devices.

## Installation

To get started with the AI Resume Builder, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/ai-resume-builder.git
   cd ai-resume-builder
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Install TypeScript and types:
   ```bash
   npm install --save-dev typescript @types/react @types/react-dom
   ```

4. (Optional) Install Tailwind CSS if not included:
   ```bash
   npm install -D tailwindcss postcss autoprefixer
   npx tailwindcss init -p
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

- Open your browser and navigate to `http://localhost:5173`.
- Fill out the form with your details.
- View the live preview of your resume.
- The PDF download feature will be available soon!
